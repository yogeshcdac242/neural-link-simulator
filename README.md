# Sci-Fi Cyber Brain Simulator

An interactive, 3D neural network simulation built with Three.js and vanilla JavaScript. 

![Cyber Brain Preview](./image11)


## Features
- **Holographic Shell**: A glowing cyan cyber-mesh constrained to an ellipsoidal brain shape.
- **Neon Neural Core**: A dense network of firing orange and red nodes with dynamic synapses.
- **Interactive Controls**: Full 3D camera controls (Zoom and Rotate) using OrbitControls.
- **Thought Pulses**: Dynamic "thoughts" fire through the network, illuminating pathways.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the provided localhost URL in your browser.
